Hands-On Session: Creating a Quarto Report

Author

Rony Rodriguez-Ramirez

Published

26 August 2024

Introduction

In this hands-on session, you will create a Quarto document that simulates a report. Youâ€™ll learn how to structure your report, integrate R code for analysis, and generate visualizations using the tidyverse. By the end of this session, youâ€™ll have a fully reproducible report that you can apply to your problem sets.

Exercise 1: Setting Up Your Quarto Document

Objective

Set up a Quarto document that will serve as the foundation for your report.

Instructions

  1. Create a New Quarto Document:
    • Open RStudio (or your preferred editor).
    • Go to File > New File > Quarto Document.
    • Choose â€œHTMLâ€ as the output format.
  2. Add Document Metadata:

    • Give your document a title, author, and date.

    • Example:

      ---
title: "TITLE"
author: "Your Name"
date: "2024-08-25"
format: html
---
  3. Save Your Document:
    • Save the file with a descriptive name, like ps_test.qmd.

Exercise 2: Writing an Introduction

Objective

Introduce the purpose of your report and the dataset you will be analyzing.

Instructions

  1. Write an Introduction Section:

    • Describe the PISA dataset and the focus of your analysis.

    • Example:

      # Introduction

This report analyzes data from the PISA dataset to explore the relationship between students' socioeconomic status and their academic performance in mathematics, reading, and science. The goal is to identify trends and disparities across different countries and years.
  2. Outline the Structure of the Report:
    • Briefly mention the different sections that will be included in the report (e.g., Data Exploration, Analysis, Visualization, Conclusion).

Exercise 4: Data Filtering and Summarizing

Objective

Filter the dataset to focus on specific countries and summarize the data to understand trends over time.

Instructions

  1. Filter the Data:

    • Focus on Canada, the United States, and Mexico.

    • Example:

      pisa_filtered <- pisa_data %>%
  filter(country %in% c("CAN", "USA", "MEX"))
  2. Summarize the Data:

    • Calculate the mean scores in mathematics, reading, and science for each country by year.

    • Example:

      pisa_summary <- pisa_filtered %>%
  group_by(country, year) %>%
  summarise(
    mean_math = mean(math, na.rm = TRUE),
    mean_read = mean(read, na.rm = TRUE),
    mean_science = mean(science, na.rm = TRUE)
  )
pisa_summary
  3. Write a Summary Section:

    • Describe the summary statistics and what they reveal about the data.

    • Example:

      # Data Summary

The summary statistics show that over the years, the average scores in mathematics, reading, and science have varied significantly across the three countries. Notably, there are consistent trends in the relative performance of each country in these subjects.

      Exercise 5: Data Visualization

Objective

Create visualizations to explore the trends in PISA scores across the selected countries.

Instructions

  1. Create a Line Plot for Math Scores:

    • Use ggplot2 to visualize the average math scores over time for each country.

    • Example:

      ggplot(pisa_summary, aes(x = year, y = mean_math, color = country)) +
  geom_line() +
  geom_point() +
  labs(
    title = "Average PISA Math Scores Over Time",
    x = "Year",
    y = "Average Math Score"
  ) +
  theme_minimal()
  2. Create Faceted Plots:

    • Create faceted plots for math, reading, and science scores.

    • Example:

      pisa_long <- pisa_summary %>%
  pivot_longer(cols = starts_with("mean_"), names_to = "subject", values_to = "score")

ggplot(pisa_long, aes(x = year, y = score, color = country)) +
  geom_line() +
  geom_point() +
  facet_wrap(~ subject) +
  labs(
    title = "Average PISA Scores Over Time by Subject",
    x = "Year",
    y = "Average Score"
  ) +
  theme_minimal()
  3. Write a Visualization Section:

    • Include the plots and describe the trends they reveal.

    • Example:

      # Data Visualization

The plots below illustrate the trends in PISA scores over time for Canada, the United States, and Mexico. Each plot is faceted by subject (mathematics, reading, and science) to allow for a clear comparison of trends across different areas of academic performance.

Exercise 6: Writing a Conclusion

Objective

Summarize your findings and discuss any patterns or trends observed in the data.

Instructions

  1. Write a Conclusion Section:

    • Discuss the key insights gained from your analysis and visualizations.

    • Example:

      # Conclusion

This analysis of the PISA dataset reveals several important trends. For instance, Canada's performance in mathematics has consistently been strong relative to the United States and Mexico. The visualizations also highlight how socioeconomic factors may influence academic performance, as seen in the varying trends across different countries.
  2. Suggest Further Analysis:
    • Mention any additional questions or analyses that could be explored in future work.

Instructions
